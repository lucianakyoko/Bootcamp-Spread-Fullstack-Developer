# AULA 1 - ECMAScript Error e DOMExpection
## ECMAScript Error:
Erros que ocorrem em tempos de execução.
É composto por:
  - Mensagem
  - Nome
  - Linha
  - Callback

  exemplo:
  ```
  Uncaught ReferenceError: a is not defined at <anonymous>:1:11
  ```

## DOMException:
Erros relacionados ao Document Object Model (DOM).
