# Operadores
## Tipos de operadores:
* Aritmética
* Atribuição
* Comparação
* Lógica
* Condicionais

### Atribuição:
= 
+=
*=
/=
%=

### Aritmética:
+
-
*
**
/
%

### Operadores de Comparação:
igual a (==)
Mesmo valor e mesmo tipo (===)
Diferente (!=)
Valor e tipos diferentes (!==)
maior que (>)
Menor que (<)
Maior ou igual (>=)
Menor ou igual (<=)

### Lógica:
"e" lógico (&&)
"ou" lógico (||)
"não" lógico (!)

### Condicionais:
(condition) ? expression1 : expression2

if(condition){
  statement (expression1)
} else {
  statement (expression2)
}
